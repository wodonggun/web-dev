# CSS 삽입 방법

```HTML
<!DOCTYPE html>
<html>
  <head>
    <style>
      h2{color:blue}
    </style>
  </head>
  <body>
    <h1 style="color:red">Hello World</h1>
    <h2>Hello world</h2>
  </body>
</html>
```

- tag 선택자
```
li{
    color:red;
    font-size: 12px;
    text-decoration:underline;
    }
```


- id 선택자
```
#select{
    font-size:50px;
  }
```
